1. INTRODUCTION

The commons-logging package provides a simple, component oriented interface (org.apache.commons.logging.Log) together with wrappers for logging systems. The user can choose at runtime which system they want to use. In addition, a small number of basic implementations are provided to allow users to use the package standalone.
commons-logging包提供简单、面向组件的接口（org.apache.commons.logging.Log）， 并包含对多种日志系统的封装，用户可在运行时选择日志系统；还提供少量基本实现，支持独立使用

commons-logging was heavily influenced by Avalon's Logkit and Log4J. The commons-logging abstraction is meant to minimize the differences between the two, and to allow a developer to not tie himself to a particular logging implementation.
commons-logging受Avalon Logkit和Log4J的强烈影响，旨在减少两者差异， 使开发者无需绑定到特定日志实现

The package does :

  • Have an API which should be as simple to use as possible
    提供尽可能简单易用的API
  • Provide support for Avalon's Logkit
    支持Avalon的Logkit日志系统
  • Provide support for JDK 1.4 logging
    支持JDK 1.4的日志功能
  • Provide support for Log4J
    支持Log4J日志系统
  • Provide support for Avalon-Framework's Logger
    支持Avalon框架的Logger
  • Provide support for LumberJack logger
    支持LumberJack日志系统
  • Provide pluggable support for other logging APIs
    支持可插拔的其他日志API
  • Provide a simple System.out logger
    提供简单的System.out日志输出

Non-goals:

  • This package will not perform logging itself, except at the most basic level.
    除最基本级别外，该包不自行执行日志记录
  • We do not seek to become a "standard" API.
    不追求成为标准API

2. DEPENDENCIES

The Logging component is dependent upon the following external components for compilation:

Of course, the user must ensure that any logging systems that they wish to use are present in the classpath when they use this component.

  • Java Development Kit (Version 1.2 or later)
    依赖Java开发工具包（JDK），版本1.2或更高
  • Apache Logging Log4j (Optional, Version 1.2.6 or later)
    可选依赖Apache Log4j，版本1.2.6或更高
  • Apache Avalon LogKit (Optional, Version 1.0.1 or later)
    可选依赖Apache Avalon LogKit，版本1.0.1或更高
  • Apache Avalon Framework (Optional, Version 4.0 or later)
    可选依赖Apache Avalon框架，版本4.0或更高
  • LumberJack
    依赖LumberJack日志系统
  • Java Development Kit (Version 1.4 or later) logging APIs (Optional)
    可选依赖JDK 1.4或更高版本的日志API
